RULE-BASED AI CHATBOT

Project Title: Enhanced Rule-Based AI Chatbot

Objective:
To create a rule-based chatbot that responds to predefined user inputs using if-else statements and runs continuously until the user exits. The chatbot also includes additional features such as date and time responses, jokes, a calculator, and a help menu to improve user interaction.

Requirements:

* Python 3.x
* Any code editor (VS Code, PyCharm, or IDLE)

Features:

1. Handles greetings such as "hi", "hello", and "hey".
2. Responds to basic questions like:

   * how are you
   * what is your name
   * who created you
   * what can you do
3. Displays the current date and time.
4. Tells jokes.
5. Includes a simple calculator for basic arithmetic operations.
6. Provides a help menu showing all available commands.
7. Recognizes exit commands such as "bye", "exit", and "quit".
8. Displays a default message for unknown inputs.
9. Runs continuously until the user chooses to exit.

How to Run:

1. Open Command Prompt or Terminal.
2. Navigate to the project folder.
3. Run the command:
   python chatbot.py
   or
   py chatbot.py

Available Commands:

* hi
* hello
* hey
* how are you
* what is your name
* who created you
* what can you do
* time
* date
* today's date
* joke
* calculator
* help
* bye
* exit
* quit

Sample Interaction:

You: hi
Bot: Hello! How can I help you?

You: time
Bot: 14:30:25

You: joke
Bot: Why do programmers prefer dark mode? Because light attracts bugs!

You: calculator
Enter first number: 10
Enter operator (+, -, *, /): *
Enter second number: 5
Result: 50

You: bye
Bot: Goodbye! Have a nice day.

Technologies Used:

* Python
* If-Else Statements
* While Loop
* Input/Output Functions
* Date and Time Module

Outcome:
This project demonstrates fundamental Artificial Intelligence concepts, control flow, decision-making logic, and user interaction by creating an enhanced rule-based chatbot with multiple features and commands.
